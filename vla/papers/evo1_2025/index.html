    Evo-1: Lightweight VLA with Preserved Semantic Alignment¶

    基本信息¶

    • 论文: Evo-1: Lightweight Vision-Language-Action Model with Preserved Semantic Alignment
    • arXiv: 2511.04555
    • 机构: SJTU + EvoMind Tech + IAAR-Shanghai + CMU + Cambridge + NTU
    • 作者: Tao Lin, Yilei Zhong, Yuxin Du, Jingjing Zhang, Jiting Liu, Yinxinyu Chen, Encheng Gu, Ziyan Liu, Hongyi Cai, Yanwen Zou, Lixing Zou, Zhaoye Zhou, Gen Li†, Bo Zhao†
    • 年份: 2025（v2: 2025.12.05）
    • 代码: https://github.com/MINT-SJTU/Evo-1
    • 方法线归属: 轻量 VLA（VLM + Cross-Modulated Diffusion Transformer，无机器人数据预训练）

    核心问题¶

    现有 VLA 模型的四大限制： 1. 参数量庞大（数十亿级），训练和推理 GPU 开销高 2. 低控制频率，限制实时交互能力 3. 端到端训练破坏 VLM 表示空间，导致过拟合和泛化能力下降 4. 依赖大规模机器人预训练数据（OXE, DROID），数据收集成本高

    核心贡献¶

    Evo-1 = InternVL3-1B（原生多模态 VLM）+ Cross-Modulated DiT + 两阶段语义保护训练

    1. 视觉-语言骨干：InternVL3-1B¶

    • 视觉编码器: InternViT-300M（从 InternViT-6B 蒸馏，逐层负余弦相似度损失）
    • 输入图像缩放至 448×448，pixel-unshuffle 下采样 4×，减少 visual token 数量
    • 语言编码器: Qwen2.5-0.5B（0.5B 参数，强语义理解）
    • 原生多模态预训练：视觉和语言表示联合学习，无需事后对齐（与 OpenVLA/Prismatic 的拼接对齐不同）
    • 仅保留前 14 层：中间层 VLM 特征具有更强视觉-语言对齐，更适合视觉运动控制
    • 对比实验：InternVL3-1B 的注意力图保持空间一致性，而 OpenVLA (Prismatic-7B) 训练后注意力退化

    2. Cross-Modulated Diffusion Transformer（动作专家）¶

    • 基于 Flow Matching（线性插值 + Beta 分布采样 τ，截断到 [0.02, 0.98]）
    • 架构为 Diffusion Transformer (DiT)，但仅用 Cross-Attention（无 Self-Attention），比 π₀/SmolVLA 的 Self+Cross 交替结构更紧凑
    • 去噪目标：预测速度场 vθ 驱动带噪动作 Aτ_t 趋向 ground-truth At

    3. Integration Module（感知-控制桥接）¶

    • Cross-Attention 将多模态表示 zt（VLM 第14层输出）与机器人本体感知 st 融合
    • 关键设计：Concat（保留完整信息）而非投影到共享空间

    4. 两阶段训练（核心创新）¶

    Stage 1 - Action Expert Alignment：冻结 VLM，只训练 Integration Module + DiT - 让动作专家学会响应 VLM 特征，不破坏 VLM 语义空间

    Stage 2 - Joint Fine-tuning：解冻（部分层）VLM 进行联合优化 - 保持 VLM 语义先验的同时，适应具体机器人任务

    架构¶

    RGB obs + Language inst → InternVL3-1B (前14层) → zt ─┐
Robot state st ─────────────────────────────────────────→ Integration Module (Cross-Attn) → Cross-Modulated DiT → Action

    推理效率¶

    • 0.77B 参数（约为 π₀ 3.5B 的 1/4.5）
    • 高推理频率 + 低 GPU 显存占用（具体数值见论文 Sec 4.3）
    • 适合消费级 GPU 部署

    关键洞察¶

    1. 语义保护的关键：两阶段训练防止 VLM 表示退化，注意力图对比直观验证了 InternVL3-1B 的空间一致性优势
    2. 中间层特征更好：VLM 第14层（非最终层）的多模态对齐特征更适合视觉运动控制
    3. Cross-only DiT：去掉 Self-Attention 使 DiT 更紧凑，实验证明性能不降
    4. 无预训练可行性：100 条演示从头训练，证明 InternVL3-1B 的原生多模态预训练已蕴含足够的视觉-语言先验
