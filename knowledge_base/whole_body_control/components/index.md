# WBC 技术组件

## 核心技术模块

### 运动重定向（Motion Retargeting）
将人类动作映射到机器人关节空间，是遥操作和运动模仿的基础。
- Retargeting Matters (2510.02252, 41引用) — 通用运动重定向框架
- OmniRetarget (2509.26633, 45引用) — 保持交互的数据生成与重定向
- 关键挑战：人体与机器人形态差异（体型、关节限制、自碰撞）

### 运动表示（Motion Representation）
如何编码动作先验，支持多样化运动生成和跟踪。
- **PHC** (2305.06456) — 持续人形控制的高效运动表示
- **UHMP** (2310.04582, 112引用) — 通用人形运动表示，物理控制
- **MaskedMimic** (2409.14393) — 遮蔽填补，统一运动先验
- **HOVER** — 多模式蒸馏，运动模仿作为通用抽象

### 策略蒸馏（Policy Distillation）
将多个专家策略蒸馏为单一通用策略。
- HOVER 多模式蒸馏：速度控制、关节控制、末端执行器控制 → 统一策略
- AMO (2505.03738) — 自适应运动优化，高度灵巧WBC
- 优势：减少策略数量，实现模式间无缝切换

### 扩散模型（Diffusion Policy）
用扩散模型生成运动轨迹，分类器引导实现测试时优化。
- **BeyondMimic** (2508.08241) — 统一潜在扩散模型，分类器引导
- 核心技术：分类器引导 → 零样本泛化到未见任务
- 可组合性：多个技能在扩散空间自然组合

### Sim-to-Real 迁移
减少仿真到真实硬件的物理差距。
- **ASAP** (2502.01143, 156引用) — 对齐仿真与真实物理
- Domain Randomization：随机化物理参数训练鲁棒策略
- 感知-动作同步：精确时序，减少延迟影响

### 全身遥操作接口
人类运动到机器人的实时映射系统。
- VR 头显 + 手柄：Open-TeleVision, OmniH2O
- 深度相机/RGB姿态估计：H2O, HumanPlus
- 外骨骼驾驶舱：HOMIE (2502.13013, 104引用)

### 大规模预训练（Foundation Models）
- **SONIC** — 运动跟踪作为预训练任务，42M参数，9000 GPU小时
- **GR00T N1** (2503.14734, 550引用) — NVIDIA开源人形基础模型
- Scaling 三轴：网络规模、数据规模、计算规模

## 下游任务接口
- **运动跟踪** → 遥操作 / VLA 指令 / 高层规划
- **速度控制** → 导航 / 路径跟随
- **末端执行器控制** → 操作任务
- **关节角控制** → 精确姿态复现
