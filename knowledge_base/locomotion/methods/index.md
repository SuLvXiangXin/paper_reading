# Locomotion 方法分类

| 方法 | 代表工作 | 说明 |
|------|---------|------|
| Domain Randomization | Tan 2018, Peng 2018 | 仿真参数随机化实现 sim-to-real |
| Teacher-Student | Lee 2020 | 特权信息蒸馏到可部署策略 |
| Rapid Motor Adaptation | RMA (Kumar 2021) | 在线推断环境参数的适应模块 |
| Adversarial Motion Priors | AMP (Peng 2021) | 判别器学习自然运动风格 |
| Massively Parallel Training | IsaacGym (Rudin 2021) | GPU 并行仿真加速训练 |
| Behavior Multiplicity | Walk These Ways (Margolis 2022) | 单策略编码步态族 |
| End-to-End Vision-to-Action | Extreme Parkour (Cheng 2023) | 像素直接到电机指令 |
| Causal Transformer | Radosavovic 2024 | 因果注意力做上下文适应 |
