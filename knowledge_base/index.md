# 研究领域总览

## 活跃领域

### VLA（视觉-语言-动作模型）
具身智能的核心方向：将视觉语言模型与机器人动作生成结合，实现通用机器人策略。

→ 详见 [vla/index.md](vla/index.md)

### Video World Model（视频世界模型）
视频世界模型：全方位提升视频生成性能（三维一致性、物理合理性、长时渲染、长时一致性、渲染速度）。

→ 详见 [video_world_model/index.md](video_world_model/index.md)

### VLN（视觉-语言导航）
视觉-语言导航：智能体根据自然语言指令在视觉环境中自主导航。六大流派：拓扑图、视频流VLM、LLM推理、导航基础模型、双系统、强化微调。
锚点：DualVLN (2512.08186), NaVid (2402.15852), StreamVLN (2507.05240)

→ 详见 [vln/index.md](vln/index.md)

### Whole-Body Control（全身控制）
类人机器人全身控制：运动模仿、全身遥操作、敏捷运动、Loco-Manipulation、基础模型。
锚点：SONIC (2511.07820), HOVER (2410.21229), BeyondMimic (2508.08241)

→ 详见 [whole_body_control/index.md](whole_body_control/index.md)

### Locomotion（机器人运动控制）
基于深度强化学习的运动控制：四足（ANYmal/Unitree）、双足/人形（Cassie/Digit/H1）、轮腿混合。
八大方向：sim-to-real、盲行、感知行走、敏捷跑酷、运动模仿、双足行走、轮腿混合、训练基础设施。

→ 详见 [locomotion/index.md](locomotion/index.md)

## 待开展

- **奖励学习 (Reward Learning)**: 从人类反馈或视频中学习奖励函数
