# RISE: Self-Improving Robot Policy with Compositional World Model (2026)

## 基本信息
- 作者: Jiazhi Yang, Kunyang Lin, Jinwei Li, Wencong Zhang, Tianwei Lin, Longyan Wu, Zhizhong Su, Hao Zhao, Ya-Qin Zhang, Li Chen, Ping Luo, Xiangyu Yue, Hongyang Li
- 机构: The Chinese University of Hong Kong, Kinetix AI, The University of Hong Kong, Shanghai Innovation Institute, Horizon Robotics, Tsinghua University
- arXiv: 2602.11075
- 链接: https://arxiv.org/abs/2602.11075
- 项目页: https://opendrivelab.com/kai0-rl

## 一句话总结
RISE 将 π0.5 类 VLA 的后训练从真实世界在线 RL 转移到可控视频世界模型中，用“可控 dynamics model + progress value model”生成带 advantage 的 imagination rollout，在三类双臂接触丰富任务上显著超过 RECAP、DAgger 和直接 PPO/DSRL。

## 问题
大规模 VLA 通过行为克隆获得基础能力，但在动态、接触丰富和长时序任务中容易因为小偏差累积失败。真实世界 on-policy RL 理论上能提升鲁棒性，却受限于安全风险、硬件成本、串行采样和人工 reset。

## 方法
- 方法线归属: VLA 后训练 / World Model RL / Advantage-conditioned policy
- 核心 idea: 用组合式世界模型替代真实环境做策略改进：dynamics model 根据多视角观测和 action chunk 预测未来，value model 评估 imagined outcome 并产生 advantage，策略再在 imagination buffer 上学习。
- 关键技术点:
  - Compositional World Model: 将未来观测生成和价值估计拆开，便于分别选择适合的视频生成架构和 VLA value backbone。
  - Controllable Dynamics Model: 从 Genie Envisioner 初始化，在 Agibot World、Galaxea 等动作标注数据上增强 action controllability，并使用 task-centric batching 提升同场景动作可控性。
  - Progress Value Model: 从 π0.5 初始化，用 temporal progress regression + TD learning 同时获得密集进度信号和失败敏感性。
  - Self-improving loop: 策略先用 RECAP 风格 offline warm-up，再在世界模型中 rollout；高/低 advantage imagined action 都进入训练，避免只模仿成功轨迹。
  - 训练期开销换推理效率: 世界模型只用于训练，部署时仍是普通 VLA 策略，无额外推理开销。

## 实验
- Benchmark: 双 7-DoF AgileX 真实机器人；Dynamic Brick Sorting、Backpack Packing、Box Closing。
- 主要结果: RISE 成功率分别达到 85%、85%、95%；相对 RECAP 的 50%、40%、60% 有明显提升；相对 π0.5 baseline 的 35%、30%、35% 提升更大。
- 对比基线: π0.5、π0.5+DAgger、π0.5+PPO、π0.5+DSRL、RECAP。

## 评价
- 优势: 把 VLA 后训练的交互瓶颈转化为可并行 imagination；value model 不只做终态奖励，而是给 chunk-level advantage；在接触丰富任务上比直接 RL 稳定。
- 局限: 仍依赖任务级真实数据和世界模型微调；视频模型误差限制 rollout horizon；实验集中在少数双臂任务，跨任务/跨具身泛化尚未证明。
- 对 VLA 领域的贡献: RISE 是 π0/π0.5 之后“VLA + world model + value-guided self-improvement”的代表路线，和纯 BC scaling、纯真实世界 RL 形成清晰对照。
