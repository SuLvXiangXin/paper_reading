# JALA: Joint-Aligned Latent Action: Towards Scalable VLA Pretraining in the Wild (2026)

## 基本信息
- 作者: Hao Luo, Ye Wang, Wanpeng Zhang, Haoqi Yuan, Yicheng Feng, Haiweng Xu, Sipeng Zheng, Zongqing Lu
- 机构: Peking University, Renmin University of China, BeingBeyond
- arXiv: 2602.21736
- 日期: 2026.02.25
- 项目: https://research.beingbeyond.com/jala

## 一句话总结
提出"联合对齐潜在动作"(JALA)框架，用**预测性嵌入与逆动力学的联合对齐**替代重建式潜在动作管线，使VLA能统一利用有标注的实验室人手视频和无标注的野外视频进行大规模预训练，在手部运动生成和机器人操作任务上均取得显著提升。

## 问题
VLA预训练受限于机器人数据稀缺。人类操作视频是天然的替代数据源，但存在**质量-多样性权衡**：
- 实验室数据集（DexYCB, ARCTIC, HOI4D等）有精确3D手部追踪但场景单一
- 野外视频（Ego4D等）多样性极高但缺乏可靠的动作标签

现有利用无标注视频的方法（如LAPA）依赖**重建式潜在动作**（IDM + FDM重建未来帧），但对细粒度手部操作而言，FDM难以准确建模手部运动的微妙变化，反而引入噪声降低潜在动作质量。

**核心问题**：如何有效融合有标注和无标注的异构人手数据源，实现可扩展的VLA预训练？

## 方法
- **方法线归属**: 
  - 预训练阶段: VLM + Action Token（基于Being-H0，用token化手部运动做预训练）
  - 后训练阶段: VLM + Flow Head（用flow matching头迁移到机器人任务）
  - 创新交叉点: 与World Model流派C (视频→IDM) 中的**LAPA等latent action方法**形成对比，提出替代重建的**联合对齐范式**

- **核心idea**: 用VLA Transformer中间层的hidden state作为**预测性嵌入(predictive embedding)**，同时与(1)逆动力学推断的潜在动作对齐、(2)真实动作标签监督，形成统一的潜在动作空间，完全绕过像素重建。

- **关键技术点**:

### 1. 联合对齐机制 (Joint Alignment)
- **Masked Chunk Prediction (MCP)**: chunk级masked token建模，用双向注意力在chunk内部建模运动关系，预测被mask的运动token → hidden state编码chunk级运动模式
- **Latent Action Perceiver (LAP)**: Perceiver模块作为IDM，从chunk起止帧(v_t, v_{t+δ})推断K个潜在动作向量{z_{i,k}}
- **联合对齐损失**: L1对齐predictive embedding h与latent action z: L_Align = Σ‖h_{i,k} - z_{i,k}‖₁
- **互补逻辑**: MCP alone只有运动标签但缺视觉动力学；LAP alone无法保证动作中心表征；联合对齐融合两者

### 2. 解耦EMA的双Perceiver架构
- **LAP + LSP (Latent State Perceiver)**: 共享Perceiver架构但异构输入
  - LAP: 输入chunk边界帧 → 生成潜在动作（用于对齐监督）
  - LSP: 输入重复的初始帧 → 注入视觉上下文到VLM嵌入空间
- **解耦EMA更新**: backbone参数从LSP→LAP传播（使LAP与预测上下文一致），query参数从LAP→LSP传播（使LSP继承动作锚定能力），α=0.999
- **消融验证**: 去掉解耦EMA (JALA w/o dec.) 导致LIBERO成功率从96.9%骤降至56.6%

### 3. 混合掩码方案
- 随机选一个chunk作主要预测目标，其内部按随机比例mask
- 前序chunk保持完整，后续chunk以5%概率mask
- 推理时多次解码每次解码~5%的token，结果集成

### 4. 混合数据训练
- 有标注数据: L = L_MCP + λ·L_Align（双重监督）
- 无标注数据: L = λ·L_Align（仅动力学对齐）
- 统一目标: L = 1_labeled · L_MCP + λ·L_Align

### 5. Flow Matching后训练
- 预训练的predictive embedding作为条件输入DiT policy head（16层，2048维）
- DiT中self-attention处理本体状态+噪声动作，cross-attention融合predictive embedding
- 第19层（共28层）hidden state效果最佳；深层过拟合数据集细节

## 数据集: UniHand-Mix
- **总量**: 7.5M样本，>2,000小时
- **实验室子集**: 5M+指令-视频-运动样本（~1,000小时实验室视频），来自ARCTIC, TACO, H2O, DexYCB, HOI4D, OakInk2, EgoDex（7个来源）
- **野外子集**: 2.5M指令-视频对（~1,123小时Ego4D），其中~10%有伪手部姿态标注
- **任务类型**: 运动生成42.5%、无运动视频29.1%、运动续写15.5%、运动描述12.9%
- **处理流程**:
  - 实验室: 手部姿态标准化→层级任务标注→指令数据生成（follow Being-H0的UniHand管线）
  - 野外: WiLoR手部检测→Gemini-2.5-Flash手部活动验证→HaWoR伪标注（可选）

## 实验

### 手部运动生成
- **评测**: Lab split + Wild split，4个指标（MPJPE, PA-MPJPE, MWTE, MDE）
- **关键结果**: 
  - Lab split: 小幅提升（JALA-dino MPJPE 7.16 vs Being-H0 7.61）
  - Wild split: 大幅提升（JALA-dino MPJPE 11.02 vs Being-H0 16.91，↓35%）
  - t-SNE可视化确认Lab和Wild数据在潜在空间中连续覆盖而非分离

### 模拟实验
- **LIBERO (two-view)**: JALA-dino 96.9%（SoTA），超越π₀ 94.2%、UniVLA 95.5%、GR00T N1.5 93.9%
- **LIBERO (single-view)**: JALA-dino 92.3%（≤3B模型SoTA），超越GR00T N1.5 92.1%、Being-H0 84.9%
- **RoboCasa Syn/Human**: JALA 27.58%/35.42%，超越GR00T N1.5 20.83%/35.17%
- **GR1 Tabletop（灵巧手）**: JALA 26.33%，大幅超越GR00T N1.5 20.41%、Being-H0 12.91%

### 真实机器人
- **平台**: Franka Research 3 + Inspire灵巧手（6-DoF）
- **任务**: Put-Three-Obj（长时序多物体）、Wipe-Board（接触操作）、Water-Plant（多指协调）
- **结果**: JALA-dino在所有任务均最优
  - Put-Three-Obj: 60.0% seen / 58.0% unseen（GR00T N1.5: 48.0%/28.0%）
  - Wipe-Board: 83.3% seen / 80.0% unseen（GR00T N1.5: 56.7%/43.3%）
  - Water-Plant: 73.3% seen（GR00T N1.5: 53.3%）
- **关键发现**: JALA对视觉分布偏移极其鲁棒（Put-Three-Obj seen→unseen仅下降2%，GR00T N1.5下降20%）

### 对比基线
- LAPA（重建式latent action）、LAPA†（用JALA backbone+数据重训LAPA）
- Being-H0（前作，无联合对齐）
- GR00T N1.5、π₀、UniVLA、OpenVLA
- 多个消融变体: JALA w/o align, JALA w/o latent, JALA w/o dec., JALA⋆, JALA-act

### 关键消融发现
1. **联合对齐是核心驱动力**: JALA-act（仅有标注数据+联合对齐）比Being-H0提升4.1%，说明方法本身有效
2. **野外数据有效扩展**: 0%→25%→50%→100%野外数据占比单调提升下游性能
3. **简单扩数据无效**: JALA w/o latent（用伪标签扩数据）反而性能下降，无结构化监督的数据扩展有害
4. **LAPA†对比控制变量**: 同backbone+同数据下LAPA†落后JALA 13+%，证明差距来自训练目标而非数据/架构
5. **训练效率**: JALA 68小时 vs LAPA†两阶段86小时（29+57），JALA节省>20%训练时间

## 评价

### 优势
1. **范式创新**: 提出"联合对齐"替代"重建"的latent action学习范式，概念清晰且实验充分
2. **真正解决了质量-多样性权衡**: 首次证明无标注野外视频可通过latent action alignment有效贡献VLA预训练，且性能单调提升
3. **与LAPA的严格对比**: 通过LAPA†消融控制了backbone/数据变量，令人信服地证明联合对齐优于重建
4. **强泛化性**: 在unseen设置下性能下降极小，说明学到了动作相关而非外观相关的表征
5. **高效**: 单阶段训练，68小时8×A800，比LAPA更快更好
6. **全面实验**: 手部生成 + 3个仿真基准 + 真实灵巧手，覆盖面广

### 局限
1. **backbone依赖Being-H0**: 方法论新颖但实现上紧耦合于Being-H0/InternVL3-2B框架，通用性有待验证
2. **预训练数据仅人手**: 7.5M人手视频，未直接包含机器人数据；与EgoScale（20K小时）、Being-H0.5（35K+小时）相比规模仍有差距
3. **真机评测较少**: 每个任务仅10次rollout，统计显著性有限
4. **层选择敏感性**: 第19层最优但第24/28层大幅下降，暗示设计选择较脆弱
5. **解耦EMA的复杂性**: 去掉后崩溃至56.6%，说明这是一个关键但脆弱的设计，可能增加调参难度

### 对VLA领域的贡献
1. **提出了重建之外的latent action学习路径**: 对LAPA、UniVLA等重建式方法提出了有力替代，在人手操作场景下优势明显
2. **验证了野外视频对VLA预训练的可扩展价值**: 与"Emergence of H2R Transfer"的发现呼应，数据量提升→性能提升在latent action框架下也成立
3. **predictive embedding作为统一接口**: 将预训练（token预测）和后训练（flow matching）通过同一中间表征连接，是一个优雅的设计模式
4. **丰富了人手数据→机器人迁移的方法谱系**: 在流派A（直接训练policy）和流派C（latent action）之间开辟了新的技术路径

## 与已有工作的关系
- **vs Being-H0 [10]**: JALA的直接前作；JALA在Being-H0基础上增加联合对齐和野外数据支持
- **vs LAPA [19]**: 核心对比对象；LAPA用IDM+FDM重建式学latent action，JALA用predictive embedding对齐式学latent action。同backbone/同数据下JALA大幅领先
- **vs UniVLA [39]**: 也用重建式latent action；UniVLA需更大backbone（>3B）和机器人数据预训练，JALA用<3B backbone仅人手数据即接近UniVLA-full性能
- **vs GR00T N1.5 [3]**: 工业级VLA基线；JALA在LIBERO、RoboCasa和真实世界均优于或持平
- **vs EgoScale/Being-H0.5**: 数据规模更大（20K/35K+小时 vs 2K小时），但方法不同；JALA聚焦latent action对齐，前两者聚焦直接策略训练/跨具身泛化

## 技术细节备注
- 骨干: InternVL3-2B（28层attention）
- 视觉编码器: DINOv3或V-JEPA2（供LAP/LSP使用）
- 运动token化: GRVQ，每chunk 128 tokens（64 wrist + 64 finger），codebook 4096
- 对齐层: 第19层（共28层）
- LAP/LSP: 2层Perceiver，共享架构，双头MLP支持双手
- 训练: 单epoch，7.5M数据，8×A800 68小时，batch 128
- 后训练DiT: 16层32头注意力，hidden 2048，4步去噪推理
