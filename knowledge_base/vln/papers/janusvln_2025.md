# JanusVLN: Dual Implicit Memory for VLN (2025)

## 基本信息
- **标题**: JanusVLN: Decoupling Semantics and Spatiality with Dual Implicit Memory
- **作者**: Shuang Zeng, Dekang Qi, Xinyuan Chang, et al.
- **arXiv**: 2509.22548
- **年份**: 2025
- **引用数**: 41
- **方法线归属**: **双通道记忆**

## 核心贡献
1. **双隐式神经记忆**：空间-几何记忆 + 视觉-语义记忆
2. 灵感来自左右脑分工理论
3. SR 提升 10.5-35.5%（vs 多模态输入方法）

## 与其他工作的关系
- 与 StreamVLN 都是双通道设计，但 JanusVLN 是隐式记忆，StreamVLN 是 SlowFast 上下文
- 提供了不同于显式地图构建的替代方案
