# InstructNav: Zero-shot Instruction Navigation (2024)

## 基本信息
- **标题**: InstructNav: Zero-shot System for Generic Instruction Navigation in Unexplored Environment
- **作者**: Yuxing Long, Wenzhe Cai, Hongchen Wang, Guanqi Zhan, Hao Dong
- **机构**: 北京大学
- **arXiv**: 2406.04882
- **年份**: 2024
- **引用数**: 123
- **方法线归属**: **LLM 推理驱动**

## 核心贡献
1. **零样本指令导航**：无需任务特定训练数据
2. 组合使用 LLM + 视觉模块实现通用指令理解
3. 在未见过的环境中直接导航

## 与其他工作的关系
- 与 NavGPT 同属 LLM-based VLN 路线
- Wenzhe Cai 也是 DualVLN/NavDP 的作者（上海 AI Lab 组）
