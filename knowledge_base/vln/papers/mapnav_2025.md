# MapNav: Annotated Semantic Maps for VLN (2025)

## 基本信息
- **标题**: MapNav: A Novel Memory Representation via Annotated Semantic Maps for VLN
- **作者**: (Zhang et al.)
- **arXiv**: 2502.13451
- **年份**: 2025
- **引用数**: 44
- **方法线归属**: **地图记忆**

## 核心贡献
1. **语义标注地图作为导航记忆**
2. 将环境观察转化为带语义标注的 2D 地图
3. VLM 直接在标注地图上做推理和规划

## 关键洞察
- 地图是人类导航的自然记忆方式
- 语义标注让 VLM 可以用语言"读"地图
