# DUET: Dual-scale Graph Transformer for VLN (2022)

## 基本信息
- **标题**: Think Global, Act Local: Dual-scale Graph Transformer for Vision-and-Language Navigation
- **作者**: Shizhe Chen, Pierre-Louis Guhur, Makarand Tapaswi, Cordelia Schmid, Ivan Laptev
- **机构**: INRIA / ENS / IIIT Hyderabad
- **arXiv**: 2202.11742
- **年份**: 2022 (CVPR 2022)
- **引用数**: 231
- **方法线归属**: **拓扑图 + Transformer**

## 核心贡献
1. **双尺度图 Transformer**：全局粗粒度（拓扑图）+ 局部细粒度（当前全景）
2. **Graph-Aware Self-Attention (GASA)**：将拓扑距离作为注意力偏置
3. **动态融合**：根据导航阶段自适应调整全局/局部权重
4. **REVERIE/SOON 上大幅 SOTA**

## 方法架构

### 全局拓扑图构建
- 无向图 G = {V, E}，导航中逐步扩展
- 三类节点：已访问、可导航（观测到未探索）、当前
- 节点视觉表示：全景图像+物体特征的 average pooling（粗粒度）
- 可导航节点：对应视角的 view embedding

### 粗粒度编码器（Coarse-scale）
- 节点嵌入 = 视觉特征 + 位置编码（相对方向/距离）+ 步骤编码
- **GASA**：`Attn = Softmax(QK^T/√d + E·W_e + b_e)V`
  - E = 成对最短路径距离矩阵，将拓扑结构注入注意力
- 多层 cross-attention（节点 ↔ 指令）+ GASA self-attention

### 细粒度编码器（Fine-scale）
- 输入：当前位置的完整全景图像+物体特征
- 标准 cross-modal Transformer（非 GASA）
- 动作空间：stop + 当前邻居节点

### 动态融合
- σ_t = Sigmoid(FFN([v0_coarse; r0_fine]))
- 最终分数 = σ·coarse + (1-σ)·fine
- 初期/末期偏局部，中间偏全局（探索需求）

## 训练策略

### 预训练
- LXMERT 初始化
- 四任务：SAP（动作预测）+ OG（物体定位）+ MLM + MRC
- REVERIE: batch 32, 100K iter, 2×P100

### 微调：PID（Pseudo Interactive Demonstrator）
- 类 DAgger：训练时用当前策略采样，PID 提供最短路径监督
- L = 0.2·L_SAP + L_PID + L_OG
- REVERIE: batch 8, 20K iter, 1×P100

## 实验结果

### REVERIE Val-Unseen / Test-Unseen
- SR **46.98 / 52.51**（HAMT: 32.95/30.40）— **+14/+22 SR**
- SPL **33.73 / 36.06**

### SOON Test-Unseen
- SR **33.44**（GBE: 12.90）— **+20.5 SR**

### R2R Val-Unseen
- SR **72**, SPL **60**（HAMT: 66/61）— SR +6, SPL 持平

## 消融实验关键结论
- 粗粒度：探索强（OSR 高），细粒度：定位准（SR/OSR 比高 93%）
- GASA 对 SPL 提升最显著（拓扑感知→更短路径）
- 动态融合优于简单平均（SPL +1.79）
- PID 微调优于 RL（SR +4.63）
- 合成数据仅在预训练有效，微调时无提升

## 局限性
1. **仅适用离散环境**：依赖预定义导航图
2. **Seen-Unseen 差距大**：SR 71.75 vs 46.98（REVERIE）
3. **物体定位有限**：到达正确位置后仅 68.43% 定位准确
4. **回溯导致轨迹变长**：SPL 无压倒性优势

## 关键技术细节
- 隐层维度 768，text encoder 9 层，coarse/fine encoder 各 4 层
- 视觉：ViT-B/16（图像）+ BUTD/ViT（物体）
- 回溯分析：Val-Unseen 48.6% 轨迹含回溯，Val-Seen 仅 13.7%
- Floyd 算法做图上最短路径
