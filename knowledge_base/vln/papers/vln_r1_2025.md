# VLN-R1: VLN via Reinforcement Fine-Tuning (2025)

## 基本信息
- **标题**: VLN-R1: Vision-Language Navigation via Reinforcement Fine-Tuning
- **作者**: Zhangyang Qi, Zhixiong Zhang, Yizhou Yu, Jiaqi Wang, Hengshuang Zhao
- **机构**: 港大, CUHK
- **arXiv**: 2506.17221
- **年份**: 2025
- **引用数**: 43
- **方法线归属**: **强化微调**

## 核心贡献
1. **首次将 GRPO（DeepSeek-R1）引入 VLN 连续导航**
2. **Time-Decayed Reward (TDR)**：近处动作权重远高于远处，指数衰减
3. **Long-Short Memory Sampling**：短期密采样 + 长期稀采样
4. **RFT 数据效率极高**：仅 20K 样本，12 小时训练，SR +5.3

## 方法架构

### 基座模型：Qwen2-VL（2B / 7B）
- 端到端：RGB 视频流 → 离散动作序列
- 四种原子动作：FORWARD(25cm), TURN-LEFT(30°), TURN-RIGHT(30°), STOP
- 每步预测未来 **6 个动作**（消融证明 6 步最优）

### Long-Short Memory Sampling
- **短期记忆**：距当前 M 步以内，间隔 δ₁ 密采样（保留近期细节）
- **长期记忆**：M 步之前，间隔 δ₂ 稀采样（保留远程上下文）
- 每实例 16 张图像，约 4.1K tokens

### GRPO（Group Relative Policy Optimization）
- 每个 prompt 生成 G=8 个响应
- 计算每个响应奖励 → 组内归一化得相对优势
- clipped ratio + KL 散度惩罚（β=0.04）
- **无需 critic model**（不同于 PPO），**无需离线偏好数据**（不同于 DPO）

### Time-Decayed Reward (TDR)
```
R_nav = Σ_{k=0}^{n-1} γ^k · I(α_{t+k} = α*_{t+k})
```
- γ < 1：近处动作正确性权重指数递减
- 第一步最重要，后续递减
- **Soft reward**（区别于 hard reward：全匹配才给 1）

## 训练数据

| 阶段 | 数据量 | 来源 |
|------|--------|------|
| SFT | 1.8M（R2R 630K + RxR 1.2M）| Habitat + MP3D |
| RFT | **仅 20K**（R2R 10K + RxR 10K）| 随机子集 |

## 训练策略

### Stage 1: SFT
- 标准交叉熵，同时监督动作标识符（A/B/C/D）和描述文本
- 学习率 5e-6，cosine，10% warmup
- 8×A800 GPU，**36 小时/epoch**

### Stage 2: RFT
- GRPO + TDR
- 学习率 1e-6，weight decay 0.01
- 每 prompt 8 个生成样本
- **仅 12 小时/epoch**，20K 样本

## 实验结果

### R2R-CE Val-Unseen

| 方法 | SR↑ | SPL↑ | NE↓ |
|------|------|------|------|
| **VLN-R1 7B** | **30.2** | **21.8** | **7.0** |
| VLN SFT 7B | 24.9 | 17.5 | 7.92 |
| VLN-R1 2B | 25.6 | 20.5 | 10.2 |
| VLN SFT 2B | 21.2 | 15.9 | 8.27 |
| R2R-CMTP（Map+Odom+Depth）| 26.4 | 22.7 | 7.90 |

- RFT 将 7B SR 从 24.9→30.2（+5.3），超过使用多传感器的方法
- **2B + RFT（25.6）超过 7B SFT（24.9）**：小模型 RL 后训练可匹配大模型

### RxR-CE Val-Unseen
- R2R SFT + **仅 10K RxR RFT** → SR **22.7%**，超过 RxR 全量训练（22.3%）
- RFT 实现高效跨域迁移

## 消融实验关键结论

| 消融 | SR | 结论 |
|------|-----|------|
| 预测 1 步 | 15.1 | 无法建模步间依赖 |
| 预测 4 步 | 21.4 | 不够 |
| **预测 6 步** | **24.9** | **最优** |
| 预测 8 步 | 22.7 | 过多反而下降 |
| 均匀采样 8 帧 | 20.8 | 忽略近期帧重要性 |
| **Long-Short Memory** | **24.9** | **最优** |
| Hard Reward | 23.8 | 不如 SFT baseline |
| Uniform Reward | 25.0 | 微弱提升 |
| 线性衰减 | 28.3 | 好于 uniform |
| **指数衰减 (TDR)** | **30.2** | **最优，+6.4 vs hard** |
| GRPO k=2 | 24.7 | 生成太少 |
| **GRPO k=8** | **30.2** | **收敛** |

## 局限性
1. **仅在仿真室内环境评估**：缺少真实世界验证
2. **离散动作空间**：仅 4 种原子动作，无法精细控制

## 与其他工作的关系
- **DeepSeek-R1 启发**：GRPO 无需 critic，数据效率极高
- **与 NaVid/StreamVLN/DualVLN 互补**：前者改架构，VLN-R1 改训练范式
- **关键洞察**: RL 后训练在 VLN 中同样有效；指数衰减奖励是关键；10K 样本 RFT 超过百万级 SFT
