# ViNT: A Foundation Model for Visual Navigation (2023)

## 基本信息
- **标题**: ViNT: A Foundation Model for Visual Navigation
- **作者**: Dhruv Shah, Ajay Sridhar, Nitish Dashora, Kyle Stachowicz, Kevin Black, Noriaki Hirose, Sergey Levine
- **机构**: UC Berkeley
- **arXiv**: 2306.14846
- **年份**: 2023 (CoRL 2023)
- **引用数**: 277
- **方法线归属**: **导航基础模型**

## 核心贡献
1. **首个视觉导航基础模型**
2. 在多种机器人形态（轮式、四足等）上预训练，展示跨机器人泛化
3. 提出 goal-conditioned navigation 的统一框架

## 方法细节
- EfficientNet 视觉编码 + Transformer temporal reasoning
- Goal-conditioned：给定目标图像，预测到达目标的动作序列
- 大规模多机器人数据预训练，支持 fine-tune 到新环境
